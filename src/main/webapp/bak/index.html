Philadelphia Eagles Defender Stats is a fan-focused web application where registered users can contribute and manage defensive statistics for Philadelphia Eagles defenders. Users will be able to enter career-based performance data such as tackles, sacks, and other metrics.

Blog

Proposed Database Table

To support this website, I propose an "Eagles Defender Stats" table. Each record represents a set of statistics entered by a user for a specific Eagles defender. The table is linked to the web_user table, allowing each user to manage their own submitted statistics.

  • stats_id - INT (primary key, auto increment)
  • player_name - VARCHAR (unique)
  • image - VARCHAR
  • total_tackles - INT (nullable)
  • solo_tackles - INT (nullable)
  • interceptions - INT (nullable)
  • sacks - DECIMAL (nullable)
  • tackles_for_loss - INT (nullable)
  • accolades (probowl, all-pro, droy) - VARCHAR
  • web_user_id - INT (foreign key)
